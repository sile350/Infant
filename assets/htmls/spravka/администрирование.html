Администрирование.

    На странице «Администрирование» осуществляется добавление (регистрация) новых, а также редактирование данных уже существующих пользователей.


Описание элементов управления


Создание (регистрация) нового пользователя.

Для регистрации нового пользователя, нужно заполнить поля под заголовком «Создать пользователя»









  • В поле «Логин»   необходимо ввести Ф.И.О. пользователя. Данные из этого поля используются для введения в поле «Логин» при входе в программу, а также прописываются в протоколы упражнений, в колонку «Специалист», для идентификации проводившего упражнение.
    • В поле «Придумайте пароль» нужно придумать и ввести пароль.
    • В поле «Подтвердите пароль» нужно точно! повторить введенный выше пароль.



      •     



       

      Если вы видите сообщение программы


      значит вы ошиблись при повторном вводе пароля. Повторите попытку. Символы, введенные для подтверждения пароля, должны ТОЧНО! совпадать с символами поля «Придумайте пароль»


      • Далее нужно присвоить пользователю уровень доступа, при помощи выпадающего меню «Уровень доступа»



          Программа имеет два уровня доступа: «Специалист» и «Администратор».

      При регистрации нового пользователя в программе, ему нужно присвоить нужный уровень доступа с соответствующими полномочиями:

      «Специалист» - имеет доступ ко всем частям программы, за исключением страницы «Администрирование», и соответственно не может добавлять новых пользователей, и/или редактировать данные ранее зарегистрированных.

      «Администратор» - имеет доступ ко всем частям и страницам программы без ограничений. В том числе имеет возможность добавлять новых пользователей, и/или редактировать данные ранее зарегистрированных.


      • Нажмите кнопку чтобы сохранить введенные выше данные в таблице «Список пользователей»


                             



      Редактирование данных пользователя

          Если нужно изменить данные, то воспользуйтесь кнопкой «Редактировать»  в строке, напротив ФИО пользователя, данные которого хотите изменить. Данные из строки выбранного пользователя переносятся в поля под заголовком «Изменить данные пользователя», где доступны для редактирования.


                                                                           


       


      • После внесенных изменений необходимо нажать кнопку сохранить, чтобы сохранить внесенные изменения!

        • Для отмены редактирования пользователя еще раз нажать кнопку редактировать



      Удаление пользователя


       

          Если необходимо удалить пользователя из программы, воспользуйтесь кнопкой «Удалить» в строке, напротив ФИО пользователя, данные которого хотите удалить. Появиться сообщение с предупреждением


      Нажатие «Продолжить» удаляет всю строку с данными выбранного пользователя.

      Нажатие «Отмена» закрывает окно предупреждения, не изменяя и не удаляя данные.


       


      Удаление данных необратимо! Будьте внимательны!
